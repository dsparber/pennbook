## List of features
Profiles, Feeds, Chat, Comments, Reactions, Affiliations, Interests, HTTPS Encryption, DNS record, 
Security Tokens (JWT), Pictures, Profile Pictures, Endless Scrolling, Group Chats, Persistent Group Chats, 
Multiple Chats with same members possible, Different Types of Posts, Automatic Posts, 
Cross Site Scripting Protection, data model support for reactions and pictures for comments, 
data model support for unlimited depth of comments (comment of comment of comment, .....), 
tokenized search, secrets stored in environment, nginx setup, custom visualizer, loading animations
